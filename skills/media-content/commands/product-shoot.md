# Product Shoot

Design a product shot for a specific product.

## Usage

```
design a product shot for [product]
create a product image for [item]
shoot a product photo of [product name]
```

## Workflow

1. Read `media-content/image-prompting/SKILL.md` for prompt crafting
2. Read `media-content/commercial-styles/SKILL.md` for product showcase patterns
3. Identify product type, material, and dimensions
4. If physical format specified: read `references/dimensions-formats.md`
5. Select product prompt template (suspended, macro, rotation, lifestyle, unboxing)
6. Choose appropriate lighting (dramatic backlight, soft spotlight, natural)
7. Define background treatment (dark moody, clean white, contextual)
8. Craft the prompt with camera/lens matched to product scale
9. Execute via `nano-banana-pro` skill
10. Save to `~/org/shared/projects/{project}/media/generated/drafts/`
11. Offer variations: different angles, lighting, or context
