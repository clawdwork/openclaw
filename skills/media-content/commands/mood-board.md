# Mood Board

Create a multi-panel mood board for a brand or concept.

## Usage

```
create a mood board for [brand/concept]
design a mood board for [theme]
make a brand mood board for [project]
```

## Workflow

1. Read `media-content/commercial-styles/SKILL.md` for mood board guidance
2. Read `references/genre-templates.md` ยง Mood Board for templates
3. Identify brand, theme, or concept
4. Determine panel count (4, 6, or 8 boxes)
5. Define color scheme and aesthetic direction
6. Craft prompt using `multi-panel mood board split into N boxes` structure
7. Include: lifestyle images, color swatches, product shots
8. Execute via `nano-banana-pro` skill
9. Save to `~/org/shared/projects/{project}/media/generated/drafts/`
10. Offer color variants using permutation approach
