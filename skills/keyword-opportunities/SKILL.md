# /keyword-opportunities

Scan an entire domain to find the fastest, highest-ROI keyword opportunities. Uses Ahrefs, Google Trends, Autocomplete, and Rank Checker to produce a prioritized action plan.

**If the argument is `help`:** Read `skills/seo/commands/keyword-opportunities.md` → find the `## Help` section → print the code block verbatim and stop.

**Full instructions:** `skills/seo/commands/keyword-opportunities.md`

## Quick Usage

```
/keyword-opportunities maxkickusa.medusajs.site
```

## What It Produces

1. **Quick Wins** — keywords at positions 11-30 with specific optimization actions
2. **Low-Hanging Fruit** — KD < 30, volume > 100, sorted by opportunity score
3. **Content Gap Matrix** — keywords competitors rank for that you don't
4. **Rising Trends** — keywords trending up >50% (early mover advantage)
5. **Long-Tail Expansion** — 50+ real autocomplete suggestions
6. **Priority Rankings** — all opportunities scored: `(Vol × CTR × Trend) / (KD × Effort)`
7. **Content Brief Queue** — top 10 articles ready to write (SOP-compliant, unique H1/meta)
8. **Revenue Projection** — top 20 opportunities × traffic × CVR × AOV
