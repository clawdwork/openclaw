## Depth of Field

| Type                          | Description                              | Use Case                       |
| ----------------------------- | ---------------------------------------- | ------------------------------ |
| **Shallow DoF** (f/1.4-2.8)   | Background heavily blurred, subject pops | Portraits, product shots       |
| **Moderate DoF** (f/4-5.6)    | Some background detail, subject clear    | Street photography, editorial  |
| **Deep DoF** (f/11-16)        | Near to far in focus                     | Landscapes, architecture       |
| **Extremely Deep DoF** (f/22) | Maximum sharpness everywhere             | Macro, technical documentation |
| **Variable DoF** (tilt-shift) | Selective focus planes, miniature effect | Creative, architectural        |
| **Infinite DoF** (pinhole)    | Entire scene softly focused              | Dreamlike, experimental        |

## Shot Types

| Shot Type             | Description                                         |
| --------------------- | --------------------------------------------------- |
| **High Angle**        | Elevated, looking down — subject appears smaller    |
| **Low Angle**         | Ground level, looking up — subject appears dominant |
| **Eye Level**         | Natural, straightforward view                       |
| **Bird's Eye**        | Overhead, top-down                                  |
| **Worm's Eye**        | Ground-up extreme low angle                         |
| **Dutch Angle**       | Tilted camera — dynamic tension                     |
| **Over-the-Shoulder** | Behind subject, focused forward                     |
| **Close-Up**          | Detailed, intimate framing                          |
| **Extreme Close-Up**  | Intense specific detail (eyes, hands, textures)     |
| **Wide Shot**         | Expansive, full scene                               |
| **Medium Shot**       | Balanced, waist-up                                  |
| **Long Shot**         | Full body + environment                             |
| **Two Shot**          | Two characters, balanced frame                      |
| **Macro**             | Extreme close-up of tiny subjects                   |
| **Contre-jour**       | Backlighting, partially silhouetted subject         |
| **Panoramic**         | Ultra-wide horizontal landscape                     |
| **Fisheye**           | Ultra-wide distorted circular perspective           |
| **Point of View**     | Character's first-person perspective                |

## Composition

| Technique               | Effect                                             |
| ----------------------- | -------------------------------------------------- |
| **Leading Lines**       | Guides viewer's focus along a path                 |
| **Horizontal Lines**    | Creates calm, stable vibe                          |
| **Vertical Lines**      | Conveys strength, elevation                        |
| **Diagonal Lines**      | Adds drama, movement, energy                       |
| **Implied Lines**       | Object sequence creates invisible path             |
| **Repetitive Lines**    | Creates rhythm and pattern                         |
| **Rule of Thirds**      | Subject at intersection points                     |
| **Natural Framing**     | Environment elements frame subject (trees, arches) |
| **Constructed Framing** | Man-made frames (doorways, windows)                |
| **Silhouette Framing**  | Dark shapes in foreground                          |
| **Shadow Framing**      | Shadows outline subject                            |
| **Symmetrical**         | Mirror-like balance                                |
| **Asymmetrical**        | Equal visual weight, different elements            |
| **Color Balance**       | Harmonious color distribution                      |
| **Tonal Balance**       | Light/dark equilibrium                             |
| **Color Blocking**      | Bold contrasting color areas                       |

## Lighting

### Direction & Quality

| Type              | Effect                                       |
| ----------------- | -------------------------------------------- |
| **Soft Light**    | Diffused, minimal shadows — flattering       |
| **Hard Light**    | Harsh, strong shadows — dramatic             |
| **Natural Light** | Sunlight, organic feel                       |
| **Ambient Light** | Broad, sourceless glow                       |
| **Backlit**       | Halo/rim light from behind subject           |
| **Sidelit**       | Side illumination — adds depth and dimension |
| **Frontlit**      | Direct front glow — minimal shadows          |
| **Toplit**        | Overhead — creates under-eye shadows         |
| **Underlit**      | From below — eerie, unsettling               |
| **Rembrandt**     | Classic triangle of light on cheek           |
| **Softbox**       | Studio diffusion — soft, even illumination   |

### Time of Day

| Time            | Quality                            |
| --------------- | ---------------------------------- |
| **Golden Hour** | Warm sunrise/sunset glow           |
| **Blue Hour**   | Cool pre-sunrise / post-sunset     |
| **Midday**      | Harsh overhead sun, strong shadows |
| **Twilight**    | Soft ambient, sun below horizon    |
| **Dawn**        | First light, delicate              |
| **Dusk**        | Last light, warm fading            |
| **Overcast**    | Diffused, shadowless               |
